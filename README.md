## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

$ npm run test <file>

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Support

包含：
logger
migration
seeder

in memory e2e test example
